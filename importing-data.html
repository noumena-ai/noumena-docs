Importing Data Into TQL¶

Before you can write any queries, you must import your raw timeseries data into TQL by constructing timelines.

from noumena.tql import *

data_path = 'gs://noumena-public/data/lethe_demo_set_randomize_users_id4_v1.3/activity*.csv'
activity = create_timeseries('activity')
activity.identified_by('user_id', 'event_time')
activity.from_files(data_path, has_csv_header=True)
